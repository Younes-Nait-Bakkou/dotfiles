

# Dotfiles
This is my dotfiles repository.

## Installation
To install this repository, run the following command:
```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/Younes-Nait-Bakkou/dotfiles/ansible-only/bin/dotfiles)"
```
